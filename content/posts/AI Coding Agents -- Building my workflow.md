# Today
If you've been around software development for any length of time, you know about coding agents. Gone now are the days of templates and snippet extensions and the like. AI is where it's at. 

However, my always preset issue is that this stuff doesn't come naturally to me. Some developers just seem to get the hang of it overnight, but having written code by hand for the better part of three decades, it feels clunky and unnatural. And I don't think I'm the only one. I need to find my workflow

## What are AI coding agents
As per chatgpt

```
AI coding agents are AI-powered software tools that can understand development tasks, write and modify code, run commands, use development tools, and iteratively work toward a goal with limited human guidance, acting more like a junior developer than a simple code autocomplete system.
```
